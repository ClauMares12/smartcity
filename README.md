# City - Smart

Plataforma de una ciudad inteligente desarrollada por BeeStudents utilizando Tecnologías para FrontEnd, UI/UX y acceso a algunas API's públicas.

## Caracteristicas

- Login y Registro de Usuarios.
- Dashboard Responsivo.
- Integración con Firebase (Paso E volutivos).
- Consumo de API's Públicas (Clima, Noticias, calidad del Aire).

## Tecnología Utilizada (Stak de Desarrollo)

- HTML5.
- CSS3.
- Bootstrap 5.
- Javascript.
- Firebase.
